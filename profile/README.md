### Repositories

| Repo | What it does | Stack |
|:-----|:-------------|:------|
| **[codejam-frontend](https://github.com/codejam-dev/codejam-frontend)** | Web app — editor, dashboard, auth flows | Next.js 15 · React 19 · TypeScript · Tailwind · Monaco |
| **[codejam-backend](https://github.com/codejam-dev/codejam-backend)** | API, auth, sandboxed code execution | Java 21 · Spring Boot 3.4 · PostgreSQL · Redis · Docker |

### What's shipped

- Multi-language execution (JS, Python, Java, C, C++, Go, Rust) in Docker sandboxes
- Monaco editor with themes, autocomplete, and editor settings
- Auth — email/password, OTP verification, Google OAuth (PKCE)
- Run history, execution metrics, share-via-URL
- Mobile-responsive playground and dashboard

### What's next

Real-time collaboration — shared rooms, live cursors, concurrent editing. Then voice/video chat and coding competitions.
