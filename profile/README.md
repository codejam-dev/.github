## What you can do today

Write code in **7 languages**, hit Run, and watch output stream back from a **Docker sandbox** — nothing runs on the host. The editor is Monaco (the same engine behind VS Code), and it works on your phone too.

## The story

CodeJam started as a simple idea: what if students could **code together** the way they study together?

Not just solve problems solo — but pair-program with a friend, compete head-to-head, practice algorithms in the same editor. A place where you open your phone, write code, run it instantly. And eventually, a platform where interviewers host live coding sessions with built-in voice and video.

The playground is live. Auth works. Execution is solid. Collaboration is next. The rest is the dream.
